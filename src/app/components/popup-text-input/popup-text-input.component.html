Die Eingabe unterstützt Markdown-Syntax. Dass heisst, dass du deinen Text formatieren kannst. Eine Vorschau der Formatierung kannst du dir anzeigen lassen, indem du auf den Knopf "Vorschau" klickst. Um eine neue Zeile zu beginnen, drücke einfach die Eingabetaste (Enter). Um einen Link einzufügen, schreibe nach einem Ausrufezeichen den Text in eckige Klammern gefolgt von der URL in runden Klammern. Beispiel: ![Beispiel-Link](https://example.com) Um Text fett zu formatieren, setze ihn zwischen drei Sternchen. Beispiel: ***fetter Text*** Um Text kursiv zu formatieren, setze ihn zwischen drei Unterstrichen. Beispiel: ___kursiver Text___ Um Text zu unterstrichen, setze ihn zwischen drei Plus-Symbole. Beispiel: +++unterstrichener Text+++ (Um Emojis einzufügen, drücke [Windows] + [.])
